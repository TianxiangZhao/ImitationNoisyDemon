# ImitationNoisyDemon

## Dependencies
* pytorch
* imitation
* imblearn
* stable_baselines3==1.5.0
* gym-minigrid==1.0.3


## Usage Instructions

1. Demonstration generation

Configurations are set inside each python file.

``> python train_expert.py ``

``> python collect_traj.py ``


2. Test model

Example 1:

``> python imit.py``

Example 2:

``> python main_Fourrooms.py``
